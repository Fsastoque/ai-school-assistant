# 🎓 Portal Educativo con Asistente Inteligente

Aplicación web desarrollada con **Streamlit** que simula un portal escolar moderno con:

* 🤖 Asistente virtual interactivo
* 🔐 Panel administrativo con login
* 📊 Dashboard con métricas
* 📰 Sección de noticias y avisos

Diseñada como solución para **Hackathon educativa**, enfocada en mejorar la interacción entre estudiantes y la institución.

## 🧠 Funcionalidades

### 🤖 Chatbot Educativo

* Responde preguntas frecuentes:

  * Horarios
  * Notas
  * Ruta escolar
* Interfaz tipo modal (experiencia moderna)

---

### 🔐 Panel Administrativo

* Login con control de sesión
* Dashboard con métricas:

  * Estudiantes
  * Cursos
  * Rutas
* Acciones simuladas:

  * Publicar avisos
  * Gestionar estudiantes
  * Ver reportes

---

### 🎨 Interfaz

* Diseño tipo portal institucional
* Componentes visuales organizados por columnas
* Modales interactivos con `st.dialog`
* Navegación simple y clara
